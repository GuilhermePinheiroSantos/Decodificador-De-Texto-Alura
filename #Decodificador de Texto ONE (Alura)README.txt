#Decodificador de Texto ONE (Alura)

Este projeto foi desenvolvido como parte do curso "Praticando lógica de programação: Challenge Decodificador de Texto" da Alura. O objetivo do curso é aplicar e reforçar conceitos fundamentais de lógica de programação através da criação de um decodificador de texto.

Funcionalidades Implementadas no projeto:

Codificação e Decodificação de Texto: O projeto permite que o usuário converta texto de acordo com a regra específica, substituindo vogais por sequências de caracteres predefinidas.

As "chaves" de criptografia que foram utilizadas são: A letra "e" é convertida para "enter" A letra "i" é convertida para "imes" A letra "a" é convertida para "ai" A letra "o" é convertida para "ober" A letra "u" é convertida para "ufat"
Requisitos:

Deve funcionar apenas com letras minúsculas;
Não devem ser utilizados letras com acentos nem caracteres especiais;
Deve ser possível converter uma palavra para a versão criptografada e também retornar uma palavra criptografada para a versão original; Por exemplo: "gato" => "gaitober" "gaitober" => "gato"
Interação com o Usuário: -Interface simples onde o usuário pode inserir o texto, realizar a codificação ou decodificação, e ver o resultado diretamente na tela. Além de, um botão para copiar o texto;

Conceitos Praticados:

Manipulação de strings;
Controle de fluxo (condicionais e loops);
Funções em JavaScript;
Manipulação do DOM com JavaScript;
Este projeto serve:

Como uma ferramenta de prática para reforçar conceitos básicos de lógica de programação;
O decodificador de texto criado permite que o usuário experimente a codificação e decodificação de mensagens e visa que somente as pessoas que sabem da regra de encriptação consiga fazer a intrepretação do texto;
Além de ser um exercício prático, o projeto também demonstra como aplicar esses conceitos em um cenário real, simulando uma aplicação que poderia ser expandida para casos de uso mais complexos;
